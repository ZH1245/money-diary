# shadcn Components Inventory

This file tracks shadcn UI components currently present in this repository and the command used to add them.

## Installed Components

- `avatar`  
  Command: `pnpm dlx shadcn@latest add avatar`
- `button`  
  Command: `pnpm dlx shadcn@latest add button`
- `calendar`  
  Command: `pnpm dlx shadcn@latest add calendar`
- `dropdown-menu`  
  Command: `pnpm dlx shadcn@latest add dropdown-menu`
- `input`

- `select`  
  Command: `pnpm dlx shadcn@latest add select`
- `separator`  
  Command: `pnpm dlx shadcn@latest add separator`
- `sheet`

- `sidebar`  
  Command: `pnpm dlx shadcn@latest add sidebar`
- `skeleton`  
  Command: `pnpm dlx shadcn@latest add skeleton`
- `tooltip`  
  Command: `pnpm dlx shadcn@latest add tooltip`

## Usage Notes

- When adding new shadcn components, append them here immediately.
- Prefer the latest CLI command:
  - `pnpm dlx shadcn@latest add <component-name>`
- If `@latest` registry fails in your environment, use:
  - `pnpm dlx shadcn@4.10.0 add <component-name>`
